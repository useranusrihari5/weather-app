# weather-app #
A responsive weather application that shows real-time weather data for any city using the OpenWeather API.
https://useranusrihari5.github.io/weather-app/

# Features

Search weather by city name,

Displays temperature in Celsius,

Shows weather condition with icon,

Humidity information,

Wind speed data,

Real-time API data,  

Responsive design.


# Technologies Used

HTML – Structure,

CSS – Styling,

JavaScript – Logic & API calls,

OpenWeather API – Weather data.


# API Setup

This project uses the OpenWeather API.

To use your own API key:

Create account at OpenWeather,

Generate API key,

Open script.js,

Replace with the API key.
